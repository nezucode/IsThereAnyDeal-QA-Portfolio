## SC06 - Compatibility Testing

### REG_SC06_TC23 - Verify that the registration page displays and functions correctly on Firefox
**Title**: Verify that the registration page displays and functions correctly on Firefox<br>
**Execution Status**: ✅ Passed <br>
**Preconditions**: 
- Firefox latest version installed on macOS Monterey
- Internet connection is active

| Steps | Action | Expected Result | Status |
| :--- | :--- | :--- |:--- |
| 1 | Open Firefox browser | The system navigates to the destination page.  | ✅ Passed |
| 2 | Navigate to https://isthereanydeal.com/#user:join | The "Create an account" page successfully loads | ✅ Passed |
| 3 | Verify page layout displays correctly | - The registration form is visually consistent. All labels, input fields, and the "Create Account" button are correctly aligned, easy to read, and fully clickable. | ✅ Passed |
| 4 | Enter a valid email to the email field  | The system accepts the valid email | ✅ Passed |
| 5 | Enter a valid password to the password field | The system accepts the valid password. The system displays the masked character in the password field (e.g., •). | ✅ Passed |
| 6 | Enter a valid password to the repeat password field | The system accepts the valid password. The system displays the masked character in the repeat password field (e.g., •). | ✅ Passed |
| 7 | Click the "Create Account" button | User is successfully registered and redirected to the "Welcome" page. The system sent a verification email to the registered email. | ✅ Passed |
